# TabBar 图标说明

此目录存放底部导航栏图标，需要准备以下图标文件（建议尺寸 81px * 81px，PNG格式）：

- `home.png` - 首页（未选中）
- `home-active.png` - 首页（选中）
- `hot.png` - 热榜（未选中）
- `hot-active.png` - 热榜（选中）
- `post.png` - 发布（未选中）
- `post-active.png` - 发布（选中）
- `mine.png` - 我的（未选中）
- `mine-active.png` - 我的（选中）

图标设计建议：
- 选中状态使用主色调 #2979FF
- 未选中状态使用灰色 #999999
- 保持简洁的线性图标风格
